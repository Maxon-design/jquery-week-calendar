jquery-week-calendar
====================

Czech
-----

For use in Czech Republic use this settings:

```
use24Hour : true,
firstDayOfWeek : 1,
daysToShow : 7,
buttonText: {
	today: "dnes",
	lastWeek : "&nbsp;&lt;&nbsp;předchozí týden&nbsp;",
	nextWeek : "&nbsp;další&nbsp;týden&nbsp;&gt;&nbsp;"
},
shortMonths : ['Led', 'Úno', 'Bře', 'Dub', 'Kvě', 'Čer', 'Čer', 'Srp', 'Zář', 'Říj', 'Lis', 'Pro'],
longMonths : ['Leden', 'Únor', 'Březen', 'Duben', 'Květen', 'Červen', 'Červenec', 'Srpen', 'Září', 'Říjen', 'Listopad', 'Prosinec'],
shortDays : ['Ne', 'Po', 'Út', 'St', 'Čt', 'Pá', 'So'],
longDays : ['Neděle', 'Pondělí', 'Úterý', 'Středa', 'Čtvrtek', 'Pátek', 'Sobota'],
timeFormat : "H:i",
dateFormat : "d.m.Y",
timeSeparator : " - ",
```
